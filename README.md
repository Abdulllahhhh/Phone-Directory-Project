# Phone Directory Project
Phone Directory project in Python as part of a course on Satr platform, 
Includes search by number, search by name, and adding new contact.

# Features
- Search by name
- Add new contact
- Search by phone number

# Language Used
Python

## Example Output

Enter phone number: 1111111111  
Amal  

Enter name: Amal  
1111111111  

Enter new name: Ali  
Enter new number: 8888888888  
Contact added
